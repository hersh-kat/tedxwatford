# Speaker photos

Drop speaker photos in this folder, then wire them up in
[`src/data/speakers.ts`](../../data/speakers.ts).

- **Shape:** square (1:1) works best for the card grid, e.g. 800×800px.
- **Format:** `.jpg` for photos, `.png` if it has transparency.
- **Naming:** kebab-case the person's name, e.g. `jane-doe.jpg`.

Astro optimises and resizes these automatically — upload the highest-quality
original you have.
